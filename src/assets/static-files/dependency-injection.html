Question 2:Scenario: You are building a RESTful API for a bookstore using
                                    Spring Boot. How would you use Dependency Injection to inject services,
                                    repositories, and configuration properties into your controllers?

                            
Answer:In a Spring Boot
                                application for a bookstore API, we can use Dependency Injection as follows:

                            
    
                                
  1. 
                                    
    Inject Services:
    
                                    
      
                                        
    • Create service classes for handling business logic and
                                            annotate them with @Service.
      • 
                                        
    • Inject service classes into the controller using
                                            constructor-based injection.
      • 
                                        
      
                                            
      
                                                java
      
                                            
      @RestController
      public class BookController {
       private final BookService bookService;

       @Autowired
       public BookController(BookService bookService) {
       this.bookService = bookService;
       }
       // ...
      }
      
                                            
      
                                        
      
                                    
    
                                    
    
                                
    2. 
                                
  2. 
                                    
    Inject Repositories:
                                    
    
                                    
      
                                        
    • Create repository interfaces for database operations
                                            and annotate them with @Repository.
      • 
                                        
    • Inject repository interfaces into service classes
                                            using constructor-based injection.
      • 
                                        
      
                                            
      
                                                java
      
                                            
      @Service
      public class BookService {
       private final BookRepository bookRepository;

       @Autowired
       public BookService(BookRepository bookRepository) {
       this.bookRepository = bookRepository;
       }
       // ...
      }
      
                                            
      
                                        
      
                                    
    
                                    
    
                                
    3. 
                                
  3. 
                                    
    Inject Configuration
                                            Properties:
    
                                    
      
                                        
    • Define a configuration properties class and annotate
                                            it with @ConfigurationProperties.
                                        
      • 
                                        
    • Inject configuration properties into the controller or
                                            service classes.
      • 
                                        
      
                                            
      
                                                java
      
                                            
      @ConfigurationProperties(prefix = "bookstore")
      public class BookstoreProperties {
       private String apiKey;
       // ...
      }
      
                                            
      
                                        
      
                                    
    
                                    
    
                                    
    java
    @RestController
public class BookController {
    private final BookstoreProperties properties;

    @Autowired
    public BookController(BookstoreProperties properties) {
        this.properties = properties;
    }
    // ...
}

    
                                
    4. 
                            

                            
By using constructor-based injection and appropriate annotations,
                                we ensure that Spring Boot manages the dependencies and provides the necessary
                                components for our controllers, services, and repositories.

                            
Question 3:Scenario: You are working on a Spring Boot application that
                                    requires external service integration. Explain how you would use Dependency
                                    Injection to inject HTTP clients or REST templates for making HTTP requests.
                            

                            
Answer:In a Spring Boot
                                application that requires external service integration, we can use Dependency Injection
                                for HTTP clients or REST templates as follows:

                            
    
                                
  1. 
                                    
    Create HTTP Client or REST
                                            Template Beans:
    
                                    
      
                                        
    • Define a bean for the HTTP client or REST template in
                                            a configuration class.
      • 
                                        
    • Configure the HTTP client or REST template with
                                            properties, authentication, or other settings.
      • 
                                        
      
                                            
      
                                                java
      
                                            
      @Configuration
      public class HttpClientConfig {
       @Value("${external-service.url}")
       private String externalServiceUrl;

       @Bean
       public RestTemplate restTemplate() {
       return new RestTemplateBuilder()
       .rootUri(externalServiceUrl)
       .build();
       }
      }
      
                                            
      
                                        
      
                                    
    
                                    
    
                                
    2. 
                                
  2. 
                                    
    Inject HTTP Client or REST
                                            Template:
    
                                    
      
                                        
    • Inject the HTTP client or REST template into your
                                            service or controller using constructor-based injection.
      • 
                                        
      
                                            
      
                                                java
      
                                            
      @Service
      public class ExternalServiceIntegrationService {
       private final RestTemplate restTemplate;

       @Autowired
       public ExternalServiceIntegrationService(RestTemplate restTemplate) {
       this.restTemplate = restTemplate;
       }
       // ...
      }
      
                                            
      
                                        
      
                                    
    
                                    
    
                                
    3. 
                            

                            
By configuring and injecting the HTTP client or REST template as
                                beans, Spring Boot manages the lifecycle and provides them as dependencies where needed
